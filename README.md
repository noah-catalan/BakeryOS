# BakeryOS: ERP para Panaderías

> **Optimización de la gestión en el sector de la panadería.**

**BakeryOS** es una aplicación web diseñada para digitalizar y centralizar la gestión operativa de panaderías artesanales. El objetivo es sustituir los métodos manuales (libretas, pizarras) por un sistema eficiente que controle inventario, producción y pedidos.

## Objetivos del Proyecto

El objetivo principal es desarrollar una aplicación web funcional (SPA) y fácil de usar para perfiles no técnicos.

*   ** Gestión de Inventario:** Control de materias primas y productos terminados en tiempo real.
*   ** Sistema de Recetas y Costes:** Escandallos automáticos para cálculo de costes y planificación.
*   ** Gestión de Pedidos:** Seguimiento de ventas a clientes recurrentes y facturación.
*   ** Asistente IA (Experimental):** Interfaz de control por lenguaje natural.

##  Stack Tecnológico

El proyecto se basa en una arquitectura moderna y *serverless*:

*   **Frontend:** React (Next.js) + Tailwind CSS.
*   **Backend & DB:** Firebase (Firestore, Auth, Cloud Functions).
*   **Lenguaje:** TypeScript.
*   **Herramientas:** Trello (Gestión), GitHub (Control de versiones).
